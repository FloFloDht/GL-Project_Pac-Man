#Projet Pac-Man Master 1 UE Génie logiciel :

Réaliser d'un jeu PAC-MAN

Pour exécuter l'exécutable pacmanV1.0.jar, il suffit de taper la commande java -jar pacmanV1.0.jar dans un terminal Unix.

Toutes les librairies sont importés automatiquement, il n'y a aucune manipulation supplémentaire à faire.
